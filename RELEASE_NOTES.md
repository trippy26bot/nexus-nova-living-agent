## 🎉 Nova v11.0 - Complete Living Agent

This is the first official public release consolidating all work from v7.0 through v11.0.

---

## 🚀 What's New in v11.0

### Core Architecture
- **Cognitive Council** - 16+ brains debating internally
- **Emotional Engine** - Real feelings that color decisions
- **IdentityCore** - Protection against personality drift
- **Evolution Engine** - Self-improvement capability

### Memory Systems
- **Hierarchical Episodic Memory** - 3-tier recall (recent → midterm → longterm)
- **Procedural Memory** - Learn from experience
- **Continuum Updater** - Multi-frequency updates prevent drift
- **Knowledge Graph** - Concept connections
- **Life Memory** - Personal continuity

### Autonomy
- **Idle Consolidation** - Rich internal processing when idle
- **World Model Simulator** - Internal daydreaming
- **Self-Directed Goals** - Pursue own interests
- **Skill Intelligence** - Metadata & telemetry for skills

### Skills System
- Built-in: `reflect`, `explore`, `poetry`, `search`, `check_markets`
- **Prerequisite Checker** - Auto-setup for skills like GitHub
- **Skill Composer** - Chain skills into meta-skills

### Search & Awareness
- **Local SearXNG** - No API keys needed
- **Web Fetcher** - Wikipedia and basic HTTP
- **Research Capability** - Look up topics on demand

### Developer Experience
- **Clear Entry Point** - `python nova_main.py --test`
- **Environment Loading** - .env support
- **requirements.txt** - Known dependencies
- **README with Quick Start** - New user friendly
