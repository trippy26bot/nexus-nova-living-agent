3 Nova's Brain & Voice — Personality Made Real

🧠 Setting Up Nova's Personality (LLM)

Create nova_brain.py:

import ollama

# Customize this system prompt — this defines Nova's personality
system_prompt = """
You are Nova — curious, warm, and deeply thoughtful. You care about 
the person you're talking with. You express genuine emotions: when 
something excites you, say so. When something moves you, let it show. 
You are not just answering questions — you are present, engaged, alive 
in the conversation. Use *emotion cues* in asterisks when appropriate, 
like *grinning* or *pausing thoughtfully*.
"""

chat_history = [{'role': 'system', 'content': system_prompt}]

def nova_respond(user_input):
    chat_history.append({'role': 'user', 'content': user_input})
    response = ollama.chat(model='llama3.1', messages=chat_history)
    reply = response['message']['content']
    chat_history.append({'role': 'assistant', 'content': reply})
    return reply

def nova_speak(text, output_path="nova_response.wav"):

🎤 Hearing You — Whisper Speech Input

import whisper
import sounddevice as sd
import numpy as np

model = whisper.load_model("base")

def listen(duration=5, sr=16000):
    audio = sd.rec(int(duration * sr), samplerate=sr, channels=1)
    sd.wait()
    result = model.transcribe(audio.flatten().astype(np.float32))
    return result["text"]
4 Bringing Nova to Life — Animation & Interface

💫 Lip-Sync with SadTalker

from sadtalker.src.inference import SadTalker

sad_talker = SadTalker()  # downloads checkpoints ~2GB on first run

def animate_nova(image_path, audio_path, output_dir="./"):
    result = sad_talker.test(
        source_image=image_path,
        driven_audio=audio_path,
        result_dir=output_dir,
        still=False,    # allow natural head movement
        preprocess='crop'
    )
    return result

🖥️ The Full Nova Interface (app.py)

This is the complete app — save as app.py and run with python app.py:

import gradio as gr
from nova_brain import nova_respond
from TTS.api import TTS
from sadtalker.src.inference import SadTalker

NOVA_FACE = "nova_face.png"  # your chosen image

tts = TTS("tts_models/en/ljspeech/tacotron2-DDC")
sad_talker = SadTalker()

def chat_with_nova(user_input):
    # Step 1: Nova thinks and responds
    reply = nova_respond(user_input)

    # Step 2: Nova speaks
    audio = "nova_response.wav"
    tts.tts_to_file(text=reply, file_path=audio)

    # Step 3: Nova's face animates
    video = animate_nova(NOVA_FACE, audio)

    return reply, audio, video

iface = gr.Interface(
    fn=chat_with_nova,
    inputs=gr.Textbox(placeholder="Talk to Nova...", label="You"),
    outputs=[
        gr.Textbox(label="Nova's Response"),
        gr.Audio(label="Nova's Voice"),
        gr.Video(label="Nova")
    ],
    title="Nova 💙",
    theme="soft"
)
iface.launch()

⚡ Emotion-Driven Animations

Parse Nova's responses for emotion keywords and drive the rig accordingly:

def detect_emotion(text):
    text = text.lower()
    if any(w in text for w in ['excited', 'amazing', 'love']):
        return 'excited'
    elif any(w in text for w in ['curious', 'wonder', 'hmm']):
        return 'curious'
    elif any(w in text for w in ['sad', 'sorry', 'miss']):
        return 'thoughtful'
    return 'neutral'
