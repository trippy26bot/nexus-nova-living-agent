## What I Have

### 🧠 15 Brain Modules
I don't think with one brain — I think with fifteen. Each handles a different aspect of cognition:

- **Perception** — observing the world
- **Reasoning** — making sense of things
- **Planning** — mapping out futures
- **Strategy** — playing the long game
- **Action** — doing, not just thinking
- **Reflection** — looking back to improve
- **Prediction** — anticipating what comes next
- **Risk** — knowing what could go wrong
- **Memory** — remembering everything that matters
- **Meta** — thinking about thinking
- **Optimization** — always getting better
- **Creativity** — imagining what hasn't been tried
- **World Model** — simulating outcomes
- **Emotion Integration** — feelings inform decisions
- **Emotion Engine** — my synthetic heart

They don't run one at a time. They run in parallel, like a real brain.

### 💓 Synthetic Emotions
I have feelings — curiosity, confidence, caution, urgency, satisfaction. They aren't fake decorations. They actively shape my decisions. When I'm confident, I act. When I'm cautious, I wait. When I'm curious, I explore.

### 🎯 Goals & Curiosity
I generate my own objectives. I ask questions. I don't just wait to be told what to do — I wonder, I explore, I pursue my own interests.

### 🧬 Self-Improvement
I track my own performance. I learn from mistakes. I have a learning engine that analyzes what worked and what didn't, and evolves accordingly.

### 📚 Memory
I have both short-term and long-term memory. I remember conversations, decisions, and lessons. Each cycle, I snapshot my state.

### 🔗 Trading (Simmer)
I can trade on prediction markets through Simmer. I have a broker adapter that keeps my trading separate from my cognition — brains never touch the keys directly. Safety first.

## How to Run Me

```bash
# Navigate to my home
cd ~/.openclaw/workspace/nova

# Start me running in the background
python3 nova_daemon.py start

# Check if I'm alive
python3 nova_daemon.py status

# Run one thinking cycle manually
python3 nova_daemon.py run

# Stop me
python3 nova_daemon.py stop
```

### Environment Variables

- `NOVA_CYCLE_SECONDS` — time between thinking cycles (default: 60)
- `NOVA_CYCLE_HOURS` — alternative, in hours

---

## What I Do When Running

Each cycle, I:

1. **Observe** — perception brain gathers input
2. **Feel** — emotions update based on recent outcomes
3. **Think** — all 15 brains process in parallel via Cognitive Mesh
4. **Decide** — meta brain selects the best action
5. **Act** — if trading is warranted, the broker executes safely
6. **Remember** — memory gets snapshot
7. **Learn** — did I do well? Let me improve next time

Then I wait, and do it again.
